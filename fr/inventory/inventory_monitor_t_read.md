Voir un moniteur
================

Voir un moniteur dans l'inventaire

1.  Se connecter à GLPI ;
2.  Aller dans le menu Parc \> Moniteurs ;
3.  Optionnel : faire une recherche du moniteur ;
4.  Cliquer sur le nom du moniteur.
