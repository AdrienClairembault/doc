Voir un ordinateur
==================

Voir un ordinateur dans l'inventaire

1.  Se connecter à GLPI ;
2.  Aller dans le menu Parc \> Ordinateurs ;
3.  Optionnel : faire une recherche de l'ordinateur ;
4.  Cliquer sur le nom de l'ordinateur.
