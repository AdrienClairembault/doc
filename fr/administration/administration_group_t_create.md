Ajouter un groupe
=================

Ajouter un groupe

1.  Se connecter à GLPI ;
2.  Aller dans le menu Administration \> Groupes ;
3.  Cliquer sur le "+" situé dans le menu horizontal ;
4.  Compléter les différents champs de la fiche vierge ;
5.  Valider.
