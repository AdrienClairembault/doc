Voir un fournisseur
===================

Voir un fournisseur dans GLPI

1.  Se connecter à GLPI ;
2.  Aller dans le menu Gestion \> Fournisseurs ;
3.  Optionnel : faire une recherche du fournisseur ;
4.  Cliquer sur le nom du fournisseur ;
