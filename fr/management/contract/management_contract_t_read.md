Voir un contrat
===============

Voir un contrat dans GLPI

1.  Se connecter à GLPI ;
2.  Aller dans le menu Gestion \> Contrats ;
3.  Optionnel : faire une recherche du contrat ;
4.  Cliquer sur le nom du contrat.
