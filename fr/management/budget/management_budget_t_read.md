Voir un budget
==============

Voir un budget dans GLPI

1.  Se connecter à GLPI ;
2.  Aller dans le menu Gestion \> Budgets ;
3.  Optionnel : faire une recherche du budget ;
4.  Cliquer sur le nom du budget.
