Ajouter un utilisateur
======================

Ajouter un utilisateur

1.  Se connecter à GLPI ;
2.  Aller dans le menu Administration \> Utilisateurs ;
3.  Cliquer sur l'icône + située dans le menu horizontal ;
4.  Compléter les différents champs de la fiche vierge ;
5.  Valider.
