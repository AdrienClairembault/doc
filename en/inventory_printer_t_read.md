Voir une imprimante
===================

Voir une imprimante dans l'inventaire

1.  Se connecter à GLPI ;
2.  Aller dans le menu Parc \> Imprimantes ;
3.  Optionnel : faire une recherche de l'imprimante ;
4.  Cliquer sur le nom de l'imprimante ;
